# Modern Blog

This is a simple, modern blog created with Markdown and JavaScript. It features a home page, an about page, and a blog section.

## Features

- Responsive design
- Project updates
- Blog posts
- Easy navigation

## Setup

1. Clone this repository
2. Open `index.md` in a Markdown viewer or convert it to HTML
3. Customize the content in `index.md`, `about.md`, and `blog.md`
4. Modify `styles.css` for custom styling
5. Update `script.js` for additional functionality

## Usage

To add new blog posts or project updates, edit the respective sections in `script.js`.
